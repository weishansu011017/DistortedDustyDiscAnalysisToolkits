# SpiralDetection.jl

SpiralDetection.jl is a placeholder package for spiral-structure analysis tools within this monorepo. Its purpose is to host the detection and post-processing workflow as a separate component, independent of the interpolation and particle I/O layers.

The code has not yet been fully reformatted or finalized. For the previous implementation, please consult the API in `result_toolkits.jl` on the `OptionalGLMakie` branch.
