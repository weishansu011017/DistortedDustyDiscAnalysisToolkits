# DistortedDustyDiscAnalysis

DistortedDustyDiscAnalysis is a Julia monorepo for SPH-based dusty-disc analysis. It is currently organised into companion packages for core interpolation and grid analysis (`Partia.jl`), particle-side Phantom data handling (`ParticleIO.jl`), tiny-matrix eigenvalue solving (`TinyEigvals.jl`), Streaming Instability analysis (`StreamingInstability.jl`), and future spiral-structure workflows (`SpiralDetection.jl`).
